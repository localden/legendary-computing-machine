## 🚀 Local Development

See [GAME_README.md](GAME_README.md) for detailed information about the game, features, and local development setup.

## 📦 Deployment

This repository is automatically deployed to GitHub Pages using GitHub Actions:
- Deploys on every push to the `main` branch
- Can be manually triggered from the Actions tab
- No build step required (static site)

The deployment workflow is located at `.github/workflows/deploy-pages.yml`.
